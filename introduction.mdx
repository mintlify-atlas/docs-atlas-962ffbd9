# Welcome to ReXGlue SDK

ReXGlue converts Xbox 360 PowerPC code into portable C++ that runs natively on modern platforms.

<Note>
  **Early Development Notice**: This project is in early development. Expect significant changes and breaking API updates as development progresses. Contributions and feedback are welcome!
</Note>

## What is ReXGlue?

ReXGlue is a **static recompilation toolkit** for Xbox 360 executables. Rather than interpreting or JIT-compiling PowerPC instructions at runtime like traditional emulators, ReXGlue generates C++ source code ahead of time that can be compiled and run natively on Windows and Linux.

ReXGlue is heavily rooted on the foundations of [Xenia](https://github.com/xenia-project), the Xbox 360 emulator. It takes a different path inspired by [XenonRecomp](https://github.com/hedge-dev/XenonRecomp) and [rexdex's recompiler](https://github.com/rexdex/recompiler) by generating C++ source code that can be compiled with standard toolchains.

## How It Works

The ReXGlue workflow consists of three main phases:

<Steps>
  <Step title="Analysis">
    ReXGlue analyzes the Xbox 360 executable (XEX format) to identify functions, data regions, and control flow patterns
  </Step>
  <Step title="Code Generation">
    PowerPC instructions are translated to equivalent C++ code with full support for register state, condition flags, and exception handling
  </Step>
  <Step title="Compilation">
    The generated C++ is compiled with your project using CMake and Clang 18+, linking against the ReXGlue runtime libraries
  </Step>
</Steps>

## Platform Support

| Platform | Architecture | Graphics Backend | Status |
|----------|-------------|------------------|--------|
| Windows  | x64         | D3D12           | ✅ Supported |
| Linux    | x64         | Vulkan          | ✅ Supported |
| macOS    | ARM64/x64   | Metal           | ❌ Not supported |

## Requirements

- **Compiler**: Clang 18 or newer
- **C++ Standard**: C++23
- **CMake**: 3.25 or newer
- **Architecture**: x64 (64-bit) only
- **Graphics**: D3D12 (Windows) or Vulkan (Linux)
